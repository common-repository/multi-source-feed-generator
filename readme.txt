== Description ==

Cronycle: RSS and Twitter feed embed for Wordpress

Embed RSS and twitter feeds into your Wordpress site which are filtered using specific key words.

[Click here to sign up for Cronycle](https://cronycle.com/wordpress "Click here to sign up for Cronycle") 

* Take Twitter feeds and RSS feeds and filter them with basic and advanced keywords
* No code required; simply create your ‘collection’ of feeds to embed using Cronycle and link it to your Wordpress site
* Complete security – uses an access code to ensure only you can use your collections on your site

[Click here for a complete guide and to sign up to Cronycle](https://cronycle.com/2015/wordpress-plugin/ "Click here for a complete guide and to sign up to Cronycle")  

== Installation ==

Download, install, and activate Cronycle Wordpress plug in. Link your Cronycle account to your wordpress using a unique 
activation code on your profile page. Whenever you click on the Cronycle icon in your post menu bar, you will be given 
a simple menu to embed a specific collection in your post or page.
